Honest scope flags

Where intent is captured but the engine still grows

A few places where the UI ships v1 honestly — surface and data shape are right, the deeper machinery layers in next. Keeps to your "accounting correctness over technical convenience" rule.

Jurisdictional CoA template packages intent only

The picker stores the entity's chosen template (Swiss SME / Lux SOPARFI / UK Standard / UAE Standard). Today only the IFRS-aligned Phase 1 Universal chart actually loads accounts — the four jurisdictional packages are signed in as intent, with a clearly-labelled "selection saved, template package shipping soon" badge. Far better than fabricating per-country charts.

Travel reimbursement posting status flag for v1

Approve → Reimburse flips a status flag and timestamps the row. The actual AP-side journal entry that pays the employee is a separate workstream — the engine kernel intentionally stays untouched in this commit.

AI-assisted document classification manual today

Documents have eight categories the user picks from. The hook for an AI suggest-on-upload is straightforward to add (similar pattern to the existing OCR draft flow); deferred so the storage / linking layer is solid first.

Travel receipts as URLs pending unification

Trip expense lines store a receipt URL today. Later step: swap for a foreign key to the Documents store so receipts dedupe and attach files directly. Same data shape on the surface — cleaner plumbing underneath.

Suggested walkthrough

A 3-minute click-through

  1. Open / — the new entry screen (the root URL).
  2. Walk a fresh signup: Welcome → Infrastructure consent → Organization Preferences (note the Beakon Currency field with the verbatim explainer) → Activity Selection.
  3. Land on /dashboard — the 8-tile launcher, gated by your activity choices. Top-right: the Beakon Currency: CHF chip.
  4. Open the Structure tile: try Thomas → Allina Holding → Allina Services Ltd. The tree paints live.
  5. Open the Accounting tile flow via Settings → Accounting Setup. Pick Swiss SME for one entity to see the honest "shipping soon" badge.
  6. Open the Travel Expense tile: new trip claim, add two receipts (try EUR + CHF to see the FX path), submit, have a colleague approve.
  7. Add an employee in Employment; submit a new trip claim from that user — the claim auto-links the Employee record. The employee detail page lists their claims.
  8. Upload a contract under Documents linked to an entity — open it, click the entity link, click Back, notice it returns to Documents (workflow context).
  9. Visit Settings → Activities, turn one off, go home — the tile is gone, the data is preserved.
