Current left-hand menu — snapshot

Snapshot of what's in the sidebar today, ahead of your suggestion.

How it behaves

  • Collapsed default on desktop is a 56 px icon rail; on hover it expands to 256 px and reveals labels and section headers as an overlay (no layout shift).
  • Top group (Home / Approvals / Financials / Anomalies) is always open and unlabelled — the “daily” surfaces.
  • Money and People default to open; Accounting, Settings, Reference, Developer default to collapsed. They’re shown expanded above so every item is visible for your review.
  • Developer only appears for staff/superuser accounts (gated by is_staff from /auth/me/).
  • A section auto-opens if the current route lives inside it (deep links never land in a closed group).
